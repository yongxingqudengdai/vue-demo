# 使用前必看
## 下载后 打开vscode 进入目录，然后 运行 cnpm i
## 然后 运行 npm run serve 
## 发现报错 ctrl + C 退出
## 根据报错依次重新安装对应的包 （所有命令 以 cnpm 开头 （因为已经换为国内源））
## 例如  Module not found: Error: Can‘t resolve qrcode
## 执行  cnpm  install --save qrcode
## 特别注意 vee-xx 包要装 2 版本
## 然后依次解决 提示缺失的包
## 最后 重新 npm run serve 
# 即可正常运行程序
